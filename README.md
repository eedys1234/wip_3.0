기술 스택
---------------------
+ Language : java

+ Framework : Spring mvc, Spring security, Spring data jpa, Spring cache
+ library : queryDSL
+ build tool : gradle 6.7.1
+ CI/CD : jenkins
+ Test framework : junit5, mockito
+ Code Quality : sonarqube, jacoco
